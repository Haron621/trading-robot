# XAUUSD+ локальный симулятор

Текущий этап — безопасный M2 из `BUILD.md`: расчёт аналитической карточки, fail-closed проверка допуска и безопасный журнал. Приложение не содержит ключей, не выполняет сетевых запросов и не создаёт ордера.

Запуск проверок: `npm.cmd test`

Локальная панель: `node src/server.js`, затем `http://127.0.0.1:3000`.

В реальной интеграции обязателен отдельный явный запуск и повторная проверка свежести данных, лимитов и статуса внешнего ордера.
