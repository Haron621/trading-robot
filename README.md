# XAUUSD+ локальный симулятор

Текущий этап — безопасный M2 из `BUILD.md`: расчёт аналитической карточки, fail-closed проверка допуска и безопасный журнал. Приложение не содержит ключей, не выполняет сетевых запросов и не создаёт ордера.

## Веб-интерфейс

Локальная панель в каталоге `webUItest/` показывает статус входа, вероятность сценария, рыночный режим, происхождение движения, условие отмены, факторы реакции и журнал решений. При текущих демонстрационных данных вход намеренно заблокирован.

Запуск:

```powershell
node src/server.js
```

Затем откройте `http://127.0.0.1:3000`. Панель доступна только на локальном компьютере.

Проверки: `node --test`

В реальной интеграции обязателен отдельный явный запуск и повторная проверка свежести данных, лимитов и статуса внешнего ордера.
